# Absensi & Clock-in

TARA mencatat kehadiran dengan **selfie**, **lokasi GPS**, dan **timestamp** untuk audit yang transparan.

## Clock-in (masuk)

1. Buka menu **Clock** di aplikasi mobile (`/m/clock`).
2. Izinkan akses **kamera** dan **lokasi** saat browser meminta.
3. Ambil **selfie** — pastikan wajah terlihat jelas.
4. Sistem mencatat waktu, koordinat GPS, dan IP Anda.
5. Konfirmasi jika clock-in berhasil.

## Clock-out (pulang)

Ulangi langkah yang sama di akhir hari kerja. Pastikan clock-out dilakukan sebelum meninggalkan area kerja (jika perusahaan menerapkan geofence).

## Selfie & watermark

Foto selfie disimpan dengan watermark berisi:

- Nama karyawan
- Tanggal, hari, dan jam
- Koordinat GPS
- Alamat IP

Ini membantu HR memverifikasi keabsahan absensi.

## Masalah umum

### Kamera hitam / tidak jalan

- Gunakan **HTTPS** (`https://tara.ralali.io`), bukan IP atau HTTP.
- Refresh halaman dan izinkan ulang kamera.
- Di iPhone: pastikan Safari, bukan browser dalam aplikasi chat.

### GPS tidak terdeteksi

- Aktifkan layanan lokasi di pengaturan perangkat.
- Berikan izin "Saat Menggunakan Aplikasi" untuk browser.

### Lupa clock-out

Hubungi supervisor atau HR untuk koreksi manual.
