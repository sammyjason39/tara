# Pengajuan Cuti

Ajukan cuti tahunan, cuti sakit, atau cuti khusus melalui TARA.

## Langkah mengajukan

1. Buka **Cuti** (`/m/leave` di mobile atau `/web/leaves` di web).
2. Pilih **tipe cuti** (tahunan, sakit, dll.).
3. Tentukan **tanggal mulai** dan **tanggal selesai**.
4. Isi **alasan** jika diperlukan.
5. Kirim pengajuan — status awal: **Menunggu persetujuan**.

## Melacak status

| Status | Artinya |
|--------|---------|
| Menunggu | Menunggu supervisor/HR |
| Disetujui | Cuti dikonfirmasi |
| Ditolak | Lihat catatan penolakan |

Anda akan menerima notifikasi saat status berubah.

## Jatah cuti

Saldo cuti tahunan ditampilkan di halaman cuti. Jika saldo tidak sesuai, hubungi HR — jangan mengajukan melebihi jatah tanpa konfirmasi.

## Tips

- Ajukan **minimal beberapa hari sebelumnya** untuk cuti panjang.
- Lampirkan surat dokter untuk cuti sakit jika kebijakan perusahaan mewajibkan.
- Cek kalender libur nasional di dashboard sebelum memilih tanggal.
