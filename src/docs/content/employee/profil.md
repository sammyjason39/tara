# Profil & Keamanan

Kelola data akun dan preferensi keamanan di halaman **Profil**.

## Informasi profil

Anda dapat melihat (dan sebagian mengedit):

- Nama lengkap dan kode karyawan
- Email dan departemen
- Role/jabatan
- Preferensi bahasa (Indonesia / English)

## Keamanan akun

### Ganti password

1. Buka **Profil** → bagian keamanan.
2. Masukkan password lama dan password baru.
3. Simpan — Anda mungkin perlu login ulang di perangkat lain.

### PIN rotasi

TARA dapat meminta **rotasi PIN** berkala untuk keamanan tambahan. Ikuti prompt di layar jika muncul.

### Password wajib diganti

Saat pertama login atau setelah reset oleh HR, sistem memaksa penggantian password sebelum melanjutkan.

## Keluar (logout)

Gunakan menu **Keluar** di pojok kanan (web) atau profil (mobile). Selalu logout di perangkat bersama.
