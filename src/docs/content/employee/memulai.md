# Memulai dengan TARA

Selamat datang di **TARA** (Total Assistance for Resources & Administration) — sistem HR untuk mengelola absensi, cuti, jadwal, dan komunikasi dengan asisten AI.

## Cara login

1. Buka [tara.ralali.io/login](https://tara.ralali.io/login) di browser (Chrome atau Safari direkomendasikan).
2. Masukkan **email kantor** dan **password** yang diberikan HR.
3. Setelah login pertama kali, Anda **wajib mengganti password** demi keamanan akun. Anda juga mungkin diminta **mengatur PIN** keamanan.

> **Tips mobile:** Tambahkan TARA ke layar utama (Add to Home Screen) agar terasa seperti aplikasi native.

## Belum pernah login? Chat TARA di WhatsApp

Jika Anda **belum pernah masuk** ke aplikasi TARA, Anda bisa minta bantuan lewat WhatsApp:

1. Pastikan nomor WhatsApp Anda sudah terdaftar dan terverifikasi di sistem HR.
2. Kirim pesan sapaan ke nomor bisnis TARA, misalnya:
   - `Halo Tara`
   - `Hai Tara`
   - `Selamat pagi Tara`
3. TARA akan membalas **dalam satu pesan** berisi:
   - Sapaan dengan nama Anda
   - Link login aplikasi
   - Email / username akun Anda
   - Password sementara (jika masih password awal dari HR)
   - Link panduan ini
   - Pengingat untuk **mengganti password** saat login pertama

Setelah berhasil login, gunakan WhatsApp TARA untuk tanya kebijakan, cuti, absensi, dan hal lainnya. Detail lengkap ada di halaman [Asisten WhatsApp](/docs/employee/whatsapp-ai).

> **Catatan:** Jika password Anda sudah diset khusus oleh HR (bukan password sementara default), TARA akan mengarahkan Anda menghubungi HR Admin untuk mendapatkan kredensial.

## Web vs Mobile

| Platform | URL | Cocok untuk |
|----------|-----|-------------|
| **Desktop / Web** | `/web` | Dashboard lengkap, lihat jadwal detail |
| **Mobile / PWA** | `/m` | Clock-in selfie, ajukan cuti cepat |

TARA otomatis mengarahkan Anda ke tampilan yang sesuai ukuran layar.

## Menu utama (karyawan)

- **Beranda** — ringkasan hari ini
- **Clock** — absen masuk/keluar dengan selfie + GPS
- **Cuti** — ajukan dan lacak pengajuan
- **Notifikasi** — pengumuman dan update status
- **SOP** — baca dokumen prosedur perusahaan
- **Profil** — data pribadi, bahasa, keamanan

## Butuh bantuan?

- Tanya kebijakan lewat **WhatsApp Asisten TARA** (jika sudah terhubung)
- Hubungi **HR** untuk masalah akun atau data karyawan
- Baca panduan lain di sidebar dokumentasi ini
