# Upload SOP

Dokumen SOP PDF adalah sumber kebenaran untuk **asisten AI WhatsApp** dan referensi karyawan.

## Upload dokumen

1. Buka **SOP** (`/web/sop`).
2. Drag & drop file **PDF** (maks. 50 MB) atau klik untuk memilih.
3. Isi **judul**, **deskripsi**, dan **kategori**.
4. Upload — sistem menyimpan file dan memulai **indexing** otomatis.

## Bulk upload

Anda dapat mengunggah hingga **20 PDF** sekaligus via upload bulk.

## Setelah upload

Sistem akan:

1. Menyimpan PDF di storage server.
2. Mengekstrak teks dari PDF.
3. Memecah menjadi chunk dan membuat embedding untuk pencarian AI.

> **Penting:** PDF hasil **scan/foto** mungkin tidak terbaca — gunakan PDF dengan teks asli (export dari Word/Google Docs).

## Re-index manual

Jika asisten AI tidak menemukan jawaban:

1. Buka **Pengaturan** → **AI Agent**.
2. Klik **Re-index SOP**.
3. Tunggu proses selesai dan uji ulang via WhatsApp.

## Menghapus dokumen

Hapus dari daftar SOP — file dan chunk indexing ikut terhapus.
