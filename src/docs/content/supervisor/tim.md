# Mengelola Tim

Supervisor dapat melihat struktur tim dan data bawahan langsung.

## Melihat bawahan

1. Login ke TARA web (`/web`).
2. Buka **Karyawan** — daftar dapat difilter ke tim Anda (sesuai konfigurasi akses).
3. Klik nama karyawan untuk detail profil, riwayat absensi, dan cuti.

## Hierarki supervisor

Sistem mendukung **rantai supervisor** — seorang karyawan memiliki satu supervisor langsung. HR mengatur assignment ini di pengaturan karyawan.

## Menugaskan supervisor (HR only)

Supervisor **tidak** dapat mengubah hierarki sendiri. Permintaan perubahan struktur tim dialihkan ke HR.

## Tips operasional

- Review absensi tim di awal minggu.
- Pastikan pengajuan cuti diproses dalam 1–2 hari kerja.
- Eskalasi ke HR untuk kasus disiplin atau koreksi data.
