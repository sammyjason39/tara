# Peran Supervisor

Sebagai **Supervisor**, Anda mengawasi tim langsung dan menyetujui pengajuan cuti bawahan.

## Akses yang Anda miliki

- Melihat daftar **bawahan langsung** dan hierarki tim
- **Menyetujui atau menolak** pengajuan cuti tim
- Memantau **absensi** karyawan di bawah Anda
- Mengakses dashboard dan laporan terbatas untuk tim

## Login

Gunakan akun dengan role **Supervisor**. Jika menu tim tidak muncul, hubungi HR untuk verifikasi assignment supervisor di data karyawan.

## Perbedaan dengan HR

| Supervisor | HR |
|------------|-----|
| Hanya tim sendiri | Seluruh perusahaan |
| Persetujuan cuti tim | Kebijakan, data master, SOP |
| Lihat absensi tim | Audit absensi semua karyawan |

## Dokumentasi terkait

Lanjut ke panduan **Mengelola Tim**, **Persetujuan Cuti**, dan **Absensi Tim** di sidebar.
