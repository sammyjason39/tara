# Persetujuan Cuti

Supervisor bertanggung jawab menyetujui atau menolak pengajuan cuti bawahan langsung.

## Alur persetujuan

1. Karyawan mengajukan cuti via TARA.
2. Notifikasi masuk ke supervisor terkait.
3. Buka **Cuti** di dashboard web.
4. Review tanggal, tipe cuti, saldo, dan alasan.
5. Pilih **Setujui** atau **Tolak** (sertakan catatan jika menolak).

## Kriteria pertimbangan

- Apakah ada **tumpang tindih** cuti dalam tim yang mengganggu operasional?
- Apakah **saldo cuti** mencukupi?
- Apakah pengajuan memenuhi **kebijakan minimal hari** (lihat SOP cuti)?

## Setelah disetujui

Status berubah menjadi **Disetujui** — karyawan dan HR mendapat notifikasi. HR dapat melakukan override jika diperlukan.

## Delegasi

Jika Anda cuti atau tidak tersedia, hubungi HR untuk reassignment approval sementara.
